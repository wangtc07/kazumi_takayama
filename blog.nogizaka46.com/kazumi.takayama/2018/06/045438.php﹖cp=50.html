2018/0617Sun高山一実（´-`）.｡oO(ただならぬ陽気

 
 
 
こんばんは＊
 
いつも応援ありがとうございます！
 
 
今日は握手会でした〜
 
仙台の皆さん、
遠方からお越し下さった皆さん、
ありがとうございました(*´ω`*)ﾉ
 
相変わらずみんないい人（ ; ; ）笑
楽しかったよー！
 
 
 
 
 
 
先日久しぶりに
乳酸キャベツ作りました。
 
丸ごと2玉！！！
 
4日間かけて発酵成功！！！
 
 
体型キープを意識していないと
すぐ太ってしまう私。
 
これなら夜遅くても罪悪感なく
むしゃむしゃ食べれる！！！
 
 
 
 
 
 
 
 
 
 
 
 
.｡.:✽・ﾟ＋.｡.:✽・ﾟ＋.｡.:✽・ﾟ＋.｡.:✽・ﾟ＋.｡.
 
 
 
 
 
 
昨日は友達の結婚式(*´-`)
 
奇跡的に参加することができました！
 
白無垢姿、
本当に本当にステキだった！
 
 
 
 
 
 
 
 
ANNA SUI×セーラームーン
ネオクイーンセレニティのイヤリング❤︎
 
 
特別な日に。
ISETANさんのコラボアイテム、
どれもとーってもステキ。
 
 
 
 
 
 
 
同級生たちにも会えて嬉しかったな〜＊
 
ジモッティーに会うと発生する
あの異様なまでの平和な空間、
なんなんだろうね（´-`）.｡oO
 
もうみんなと出会ってから
20年くらいになるよー
 
 
放課後によく回転寿司に行ってた
メンツで同じ円卓だったんだけど、
今日は高級和食だねなんて話しながら
料理を食べてました！笑
 
あの頃と環境は変わったけれど
感覚は変わってない。
 
...本当は、感覚もちょっと
変わってたのかもしれないけど
また戻ることができた気がするよ。
 
 
「そして父になる」を見て
ちょっぴりセンチメンタルになりながら
書いていた文章です。笑
 
 
 
 
お腹が大きな私の親友。
手を当てたら元気に動いたの！
 
母子ともにどうか健康で、ね。
 
 
 
 
 
 
 
 
 
 
.｡.:✽・ﾟ＋.｡.:✽・ﾟ＋.｡.:✽・ﾟ＋.｡.:✽・ﾟ＋.｡
 



セラミュ、MOONチームは
3日間空くことになるのですが、
公演のことを考えない日はありません。
 
結婚式でも、巫女さんの動きを
自然と目で追っちゃってた:;(∩´﹏`∩);:
 
 
 
今日の握手会もセラミュの感想を
沢山いただけて嬉しかったです！！
 
 
 
 
なるちゃん役の山内優花ちゃん！！！
 
めちゃめちゃしっかりしていて
演技もすっごーくうまい（ ; ; ）
声が透き通っててさ、美しいの。
 
あとね、じょーとの会話が面白い！笑
返しがうまいのよ！
ノリが似てるから楽しいよね！
 
 
演技に対しても一緒に考えてくれたり、
優しくて頼もしくて...（ ; ; ）
 
ウテナのアンシーを見た時からずっと
ステキな女優さんだなぁと思ってました！
同じ年！これからもずっと尊敬！！！
 
 
 
 
キャストの皆さんと、
もうすっかり仲良くなったよ*´ω`*
(私は勝手にそう思ってる)
 
載せたい写真もたくさん！
また定期的に更新しますね！
 
いつも仲良くしてくれて、
ステージでは助けてくれて、
かっこいい背中を見せてくれて、
ありがとうございます！！！
 
 
6月はあと5公演。
 
よろしくお願いします(*´ｰ`* )
 
 
 
 
 
.｡.:✽・ﾟ＋.｡.:✽・ﾟ＋.｡.:✽・ﾟ＋.｡.:✽・ﾟ＋.｡.

 
 
 
メンバーのみんなと、久しぶりに収録！
 
 
 
後ろのまちゅが絶妙にかわゆい。
 
この日、
乃木ジャーであることをして
全身筋肉痛になりましたとさ。
 
 
 
 
 
 
 
 
 
 
最後に【告知】です。
 
 
○6月24日
   →TBSさん「東大王」
 
こちらも休演日に収録！
この前書いてた大好きな番組とは
東大王のことでした！
 
相変わらず脳みそはからっぽですが、
クイズは楽しんでます！笑
ぜひ見てくださいねー！！！
 
 
 
 
 
 
 
 
 
ではでは！
 
今日はこのへんで(*´ω`*)ﾉ
 
沢山書いちゃった！ごめんごめん！
 
 
読んでくださりありがとうございました！
